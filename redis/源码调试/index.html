    源码调试

    1.调试准备¶

    • Redis6.0源代码：https://github.com/redis/redis/tree/6.0
    • Vscode软件
    • gdb（自行百度安装）

    2.配置¶

    1. 用vscode打开项目源码，点击依次点击 运行-->添加配置，会打开一个lanuch.json文件，按照如下内容修改配置：
    {
    // 使用 IntelliSense 了解相关属性。 
    // 悬停以查看现有属性的描述。
    // 欲了解更多信息，请访问: https://go.microsoft.com/fwlink/?linkid=830387
    "version": "0.2.0",
    "configurations": [
        {
            "name": "(gdb) 启动",
            "preLaunchTask": "Build",  // 新增
            "type": "cppdbg",
            "request": "launch",
            "program": "${workspaceFolder}/src/redis-server",  // 更改
            "args": [],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}",
            "environment": [],
            "externalConsole": false,
            "MIMode": "gdb",
            "setupCommands": [
                {
                    "description": "为 gdb 启用整齐打印",
                    "text": "-enable-pretty-printing",
                    "ignoreFailures": true
                }
            ]
        }
    ]
}
    1. 新建文件.vscode/tasks.json,写入如下内容：
    {
    "version": "2.0.0",
    "tasks": [
        {
            "label": "Build", 
            "type": "shell", 
            "command": "make",
            "args": [
                "CFLAGS=\"-g -O0\""
            ]
        }
    ]
    }
